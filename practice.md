# Lecture 1 Learning
I learned that in JavaScript, var was used originially to define a variable and older texts still use this method to define a variable in their code. However, var is function scoped instead of block scoped, so no error will be returned even if the variable is defined outside of a block. To avoid this, const is used instead. 

# REPL Snippet
```
0.1 + 0.2
let x;
typeof null
```
